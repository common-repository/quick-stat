== Description ==

Displays quick statistics of your blog about its PageRank, Alexa Rank, and BackLinks right on the dashboard page to help you monitor your blog's rank.

You will see additional box on dashboard page once the plugin is activated. It uses InfoLinks's and Popurius's service to checks your PageRank, Alexa Rank, and BackLinks and show it on a table.


== Installation ==


1. Upload the zipped file to yoursite/wp-content/plugins
1. Activate the plugin through the 'Plugins' menu in WordPress
